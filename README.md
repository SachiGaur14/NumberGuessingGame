# Number Guessing Game

It is a simple console-based number guessing game implemented in Java. Test your guessing skills and try to guess the correct number within 5 attempts.

## Table of Contents
- [Features](#features)
- [Prerequisites](#prerequisites)
- [How to Play](#how-to-play)
- [Contributing](#contributing)

## Features

- Random number generated by the computer between 1 and 100.
- 5 attempts to guess the correct number.
- User-friendly feedback on each guess.
- Win or lose the game based on the number of attempts.

## Prerequisites

To run this game, you need to have Java installed on your computer.

## How to Play

The computer selects a random number between 1 and 100.
You are given 5 attempts to guess the correct number.
After each guess, the game will provide feedback on whether the actual number is greater or less than your guess.
If you guess the number correctly within 5 attempts, you win!
If you exhaust all your attempts without guessing the correct number, you lose.
